# 🌱 AgriAI

AI-powered crop disease detection platform for farmers and agricultural professionals.

## 🚀 Live Demo

👉 https://huggingface.co/spaces/PARTH0076/agri-ai

## 📂 Source Code

👉 https://github.com/Parth070711/AgriAI

## Features

* Upload crop leaf images
* AI disease detection
* Confidence score
* Disease severity analysis
* Health score calculation
* Treatment recommendations
* Rice disease detection
* Leaf validation (Leaf vs Non-Leaf)

## Supported Crops

* 🍅 Tomato
* 🥔 Potato
* 🫑 Bell Pepper
* 🌾 Rice

## Tech Stack

### Frontend

* React
* Vite

### Backend

* FastAPI
* PyTorch

### AI Models

* ResNet18 Disease Detection Model
* Leaf Detector Model
* PlantVillage Dataset

## Run Frontend

```bash
cd frontend
npm install
npm run dev
```

## Run Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```
