\# 🌱 AgriAI



AI-powered crop disease detection platform.



\## Features



\- Upload crop leaf images

\- AI disease detection

\- Confidence score

\- Disease severity

\- Health score

\- Treatment recommendations



\## Tech Stack



\### Frontend

\- React

\- Vite



\### Backend

\- FastAPI

\- PyTorch



\### AI Model

\- ResNet18

\- PlantVillage Dataset



\## Run Frontend



cd frontend

npm install

npm run dev



\## Run Backend



cd backend

uvicorn main:app --reload
